Matchstiq

Vendor Management System

The problem: Both clients and recruiters need a data-driven, digital solution for job order fulfillment that is quick and easy to use.

The solution: Matchstiq is a simple online tool with collaborative job order management, comprehensive analytics and in-app activity tracking for both hiring managers and recruiters.

  • Content: Jen Hyde
  • Design + Front-end: Jason Kellum
  • Development: Mark Montroy

In-App Screens

When I joined the Matchstiq team, there was already a v1 out in the world. I was brought on to help with the next version, as the team was making pivots& in the product. My first tasks were to update the landing, authentication, and resources pages.

We then move onto the interior. Starting with a custom dashboard giving recruiters an overview of information, in an easily digestible format.

Wireframe Prototype

We partnered up with Soliant Health to run a 3 month initiative to see if we could use the power behind Matchstiq to create a better system for hiring and managing contract nurse positions and applicants.

We created an application workflow which provides recruiters incentives to provide quality and timely applicants, through a tiered point system.

Our walkaway deliverable was a wireframe mockup using Balsamiq then uploading to InVision to create a clickable prototype to share back with the Soliant stakeholders.

Support Materials

Along with the web app, we created quickly consumable, support materials to provide internal stakeholders and sales teams; including: FAQ's, Guides, and Presentations.
