RésuméBot

Format résumés faster

RésuméBot is a tool to help recruiters format resumes faster. Tasks like organizing content, adding bullets, and styling text are automated and completed within seconds.

The concept for this promotional t-shirt was to represent the animation of consuming a resume, while keeping it on a printed medium; changing expressions as you read the text.
